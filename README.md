## Project Structure

- `frontend/` - public site pages, assets, and static UI
  - `index.html` - homepage
  - `about.html` - academy information
  - `courses.html` - available courses
  - `contact.html` - contact information
  - `login.html` - student login
  - `register.html` - student registration
  - `assets/css/` - stylesheets
  - `assets/js/` - JavaScript files
  - `assets/images/` - image assets

- `backend/` - API, auth, config, and middleware code
  - `config/` - configuration files
  - `api/` - API endpoints
  - `auth/` - authentication logic
  - `middleware/` - middleware functions

- `admin/` - admin dashboard and management
  - `dashboard.html` - admin dashboard
  - `add_course.html` - add new courses

- `student/` - student dashboard and course access
  - `dashboard.html` - student dashboard

- `database/` - schema, seeds, and database scripts
